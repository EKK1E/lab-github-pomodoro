# Pomodoro App – GitHub Workflow Lab

## Team
- Leader / JS Logic: EKK1E
- Frontend / QA: Anon1mys

## Project Goal
The goal of this project is to demonstrate team collaboration using GitHub workflow.

## Features
- Pomodoro timer (start / pause / reset)
- Work modes: 25/5, 15/3, 50/10
- Task list with localStorage persistence

## Technologies
- HTML
- CSS
- JavaScript
- GitHub (Issues, Pull Requests)

## How to run
Open `index.html` in a web browser.

## Workflow
Work was organized using:
Issues → branches → Pull Requests → code review → merge.
